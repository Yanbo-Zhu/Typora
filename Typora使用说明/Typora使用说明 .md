# 文本格式

## 代码块

- 使用`包裹的内容将会以代码样式显示，例如

```bash
使用`printf()`
```

则会产生`printf()`样式。

- 输入`~~~`或者```然后回车，可以输入代码块，并且可以选择代码的语言。例如：

```java
 public Class HelloWorld{
   System.out.println("Hello World!");
 }
```
